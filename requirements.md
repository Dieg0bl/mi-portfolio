# Requisitos del Proyecto

## Historias de Usuario
1. **Como visitante**, quiero ver una sección de "Sobre mí" para conocer al autor del portafolio.
2. **Como reclutador**, quiero descargar el CV en formato PDF para revisarlo offline.
3. **Como usuario**, quiero un diseño responsivo para navegar cómodamente desde cualquier dispositivo.

## Requisitos No Funcionales
- **Seguridad**: Proteger los datos del formulario de contacto.
- **Rendimiento**: Tiempo de carga menor a 2 segundos.
- **Escalabilidad**: Capacidad para manejar hasta 10,000 visitas mensuales.
